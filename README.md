# springboot集成springcloud
springcloud是架设在springboot上的微服务分布式系统管理组件，
可以在应用开发过程中涉及到的配置管理、
服务发现、熔断器、智能路由、微代理、控制总线、
分布式会话和集群状态管理等操作提供了一种简单的开发方式
